# Swapr LP Zapping

This repository contains Zap contract which allows Swapr user to convert single-asset coins straight into LP tokens.

## Clone Repository

`git clone https://github.com/SwaprDAO/swapr-lp-zapping.git`

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`

## Flatten Contracts

`yarn flattener`
